Reference

CSS Style sheets

display: flex อยู่ใน CSS Flexbox ซึ่งช่วยให้เราสามารถจัดเรียง element ต่าง ๆ ที่อยู่ใน container เดียวกันได้แบบ responsive
ตามที่เราต้องการ
flex อยู่ใน CSS Flex Items ซึ่งเป็นการเขียนแบบ shorthand ของ flex-grow, flex-shrink และ flex-basis
• flex-grow ใช้กำหนดขนาดของแต่ละ item ว่าให้ขยายขนาดเท่าไหร่ใน container
• flex-shrink ใช้กำหนดขนาดของแต่ละ item ว่าให้ย่อขนาดหรือหดตัวเท่าไหร่ใน container
• flex-basis ใช้กำหนดความยาวเริ่มต้นของ item ใน container
flex-warp อยู่ใน CSS Flex Container ซึ่งกำหนดว่าถ้า element ต่าง ๆ มีการบีบอัดกันเกิดขึ้นเนื่องจาก Container มีขนาดเล็ก
จะให้มีการ Wrap หรือขึ้นบรรทัดใหม่ใน Element ส่วนที่เกินมา
transition ช่วยให้การเปลี่ยนแปลงค่าระหว่าง จุดเริ่มต้นและจุดจบเป็นไปอย่าง smooth มากขึ้น
transform การที่จะใส่เอฟเฟ็กต่างๆให้แก่ element อย่างเช่นการย้าย element ไปตำแหน่งที่กำหนด, การหมุน element เป็นต้น
box-shadow การทำเงาให้กับ element ใดๆ ที่เราต้องการ
-ms CSS ที่ใช้ในการแสดงผลเว็บ รองรับการใช้ใน Internet Explorer
-webkit CSS ที่ใช้ในการแสดงผลเว็บ รองรับการใช้ใน Safari, Chrome และ Opera
z-index การใช้จัดเรียง element ที่ซ้อนกันและต้องเป็น element ที่ถูกกำหนด position ไว้เท่านั้น
justify-content กำหนดการจัดตำแหน่งของ items เมื่อเทียบกับทิศทางของแกนหลัก
align-items กำหนดการจัดตำแหน่งของ items เมื่อเทียบกับทิศตรงกันข้ามของแกนหลัก
@media Media Query เป็นความสามารถหนึ่งของ CSS3 ใช้ทำเว็บไซต์ Responsive โดยกำหนดขนาดหน้าจอของ Device หรือ
อุปกรณ์แสดงผล ใช้คำสั่ง Media Query โดยจะเขียนคำสั่ง CSS ปกติ และเขียนคำสั่งลงไปในเงื่อนไข หลักการของคำสั่งตัวนี้
คือ เอาหน้าจอของอุปกรณ์มากำหนด
